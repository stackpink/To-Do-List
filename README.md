## Tech Stack
1.Frontend: HTML + CSS (custom grid layout) + JavaScript (manual DOM manipulation, fetch API)
2.Backend: Node.js + Express
3.Database: MongoDB + Mongoose
4.Authentication: JWT + bcrypt
5.Validation: Zod schemas

## Features
1. SignUp / SignIn
2. Create a new task - " +procrastinate later " button
3. Remove or Slay a task when done
4. Edit or/while regreting
5. view your list of tasks
